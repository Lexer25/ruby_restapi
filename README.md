Установка программы
1. Запустите установочный файл
2. Найдите и запустите файл ArtonitRestApi_Setup_1.0.X.exe.
3. Выберите путь установки, по умолчанию программа установится в C:\ArtonitRestAPI, при необходимости измените путь и нажмите "Далее".
4. Опционально: Установка как службы
	4.1.Отметьте "Установить как службу", если требуется, нажмите "Далее".
5. Проверьте параметры установки, нажмите "Установить", после завершения нажмите "Завершить".



Настройка подключения к базе данных:
Путь к базе данных находится в файле конфигурации "C:\ArtonitRestAPI\appsettings.json"
Параметр "DatabaseConnectionString".
путь по умолчанию: "C:\\Program Files (x86)\\Cardsoft\\DuoSE\\Access\\", после папки Access необходимо вставить название файла с расширение "GDB".

Инструкция по запуску:
1. Если была произведена установка службы, то для запуска API необходимо ввести следующий адрес "http://localhost:5056/swagger/index.html"
2. Если была произведена установка без службы, то для запуска необходимо перейти в папку "C:\ArtonitRestAPI", в ней найти файл "ArtonitRESTAPI.exe" и запустить его, 
после чего ввести в строку браузера "http://localhost:5056/swagger/index.html".
